## 🚀 3 خطوات فقط:

### 1. تأكد من الرفع على GitHub ✅

المشروع مرفوع بالفعل! ✅

---

### 2. اذهب إلى Netlify 🌐

**الرابط:** https://app.netlify.com

1. سجّل دخول (استخدم GitHub)
2. اضغط **"Add new site"**
3. اختر **"Import an existing project"**
4. اختر **"Deploy with GitHub"**
5. ابحث عن: `omier-365/amarcosmatics`
6. اضغط على المشروع

---

### 3. إعدادات النشر 🛠️

Netlify سيكتشف الإعدادات تلقائياً:

```
Build command: npm run build
Publish directory: .next
Branch: main
```

**لا تغير شيء!** فقط اضغط **"Deploy site"**

---

## ⏳ انتظر النشر (2-3 دقائق)

Netlify سيقوم بـ:
1. ✅ تثبيت المكتبات
2. ✅ بناء المشروع
3. ✅ نشر الموقع
4. ✅ إعطائك URL مثل: `https://random-name.netlify.app`

---

## 🎯 (اختياري) تخصيص الاسم

1. اذهب إلى **Site settings** → **Site details**
2. اضغط **"Change site name"**
3. اختر اسم مثل: `ammar-beauty`
4. الموقع سيصبح: `https://ammar-beauty.netlify.app`

---

## ⚙️ (اختياري) إضافة المتغيرات البيئية

1. اذهب إلى **Site settings** → **Environment variables**
2. اضغط **"Add a variable"**
3. أضف:

```
NEXT_PUBLIC_SITE_URL = https://your-site.netlify.app
NEXT_PUBLIC_WHATSAPP_NUMBER = +201202119847
```

4. اضغط **Save**
5. أعد النشر: **Deploys** → **Trigger deploy**
