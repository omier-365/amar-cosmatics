## ❓ مشاكل شائعة

### "npm is not recognized"
**الحل:** ثبّت Node.js من https://nodejs.org/

### "Module not found"
**الحل:** شغّل `npm install` أولاً

### "Port 3000 is already in use"
**الحل:** أغلق البرامج الأخرى أو استخدم port آخر:
```bash
npm run dev -- -p 3001
```
