# 📦 دليل تثبيت Node.js - موقع عَمّار

## 🎯 المتطلب الأساسي

لتشغيل موقع **عَمّار**، يجب تثبيت **Node.js** على جهازك.

---

## ✅ الخطوة 1: تحميل Node.js

### الطريقة الأسهل (موصى بها):

1. اذهب إلى الموقع الرسمي:
   ```
   https://nodejs.org/
   ```

2. ستجد زرّين:
   - **LTS** (Long Term Support) - موصى به ✅
   - Current (آخر إصدار)

3. اضغط على **LTS** لتحميله

---

## 💿 الخطوة 2: التثبيت على Windows

### خطوات بسيطة:

1. **شغّل الملف المحمّل:**
   - `node-v18.x.x-x64.msi` (أو الإصدار الأحدث)

2. **اتبع معالج التثبيت:**
   - اضغط "Next" ✅
   - وافق على الشروط ✅
   - اختر المسار الافتراضي ✅
   - اضغط "Install" ✅
   - انتظر حتى انتهاء التثبيت ⏳

3. **أنهِ التثبيت:**
   - اضغط "Finish" ✅

### ⏱️ الوقت المتوقع:
2-3 دقائق فقط!

---

## 🔍 الخطوة 3: التحقق من التثبيت

### افتح PowerShell أو CMD جديد:

```bash
# تحقق من Node.js
node --version
# يجب أن يظهر: v18.x.x أو أعلى

# تحقق من npm
npm --version
# يجب أن يظهر: 9.x.x أو أعلى
```

### ✅ إذا ظهرت الأرقام:
**مبروك! Node.js مثبت بنجاح!** 🎉

### ❌ إذا ظهر خطأ:
1. أعد فتح PowerShell/CMD بصلاحيات المدير
2. أعد تشغيل الكمبيوتر
3. جرّب الأمر مرة أخرى

---

## 🚀 الخطوة 4: تشغيل موقع عَمّار

الآن بعد تثبيت Node.js:

### 1. افتح PowerShell في مجلد المشروع:
```bash
cd d:\عمار
```

### 2. ثبّت المكتبات:
```bash
npm install
```
⏱️ **الوقت:** 1-2 دقيقة

### 3. شغّل المشروع:
```bash
npm run dev
```

### 4. افتح المتصفح:
```
http://localhost:3000
```

---

## 🎊 تهانينا!

موقع عَمّار يعمل الآن! 💄✨

---

## ❓ مشاكل شائعة

### المشكلة: "node is not recognized"

**الحل:**
1. أعد تشغيل PowerShell/CMD
2. أعد تشغيل الكمبيوتر
3. تأكد من تثبيت Node.js بنجاح

---

### المشكلة: "Permission denied"

**الحل:**
افتح PowerShell بصلاحيات المدير:
- اضغط بزر الماوس الأيمن على PowerShell
- اختر "Run as Administrator"

---

### المشكلة: "npm install" بطيء

**الحل:**
- انتظر، هذا طبيعي في أول مرة
- قد يستغرق 2-5 دقائق حسب سرعة النت

---

## 📝 ملخص الأوامر

```bash
# 1. التحقق من التثبيت

# 4. تشغيل المشروع
npm run dev

# 5. للبناء (Build)

# 6. للتشغيل الإنتاجي

```
