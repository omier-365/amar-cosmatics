# دليل الإعداد السريع - موقع عَمّار

## 📋 الخطوات السريعة

### 1. تثبيت المكتبات

```bash
npm install
```

إذا لم يكن لديك npm مثبت، قم بتحميل [Node.js](https://nodejs.org/) أولاً.

---

### 2. إعداد ملف البيئة

انسخ ملف المثال:

```bash
# Windows
copy .env.example .env.local

# Mac/Linux
cp .env.example .env.local
```

افتح `.env.local` وأضف البيانات:

```env
NEXT_PUBLIC_WHATSAPP_NUMBER=+9647XXXXXXXXX
```

*(يمكنك تخطي Supabase في البداية - الموقع يعمل بالبيانات التجريبية)*

---

### 3. تشغيل الموقع

```bash
npm run dev
```

افتح المتصفح على: **http://localhost:3000**

🎉 **تهانينا! الموقع يعمل الآن**

---

## 🗄️ إعداد قاعدة البيانات (اختياري)

إذا أردت استخدام قاعدة بيانات حقيقية:

### 1. إنشاء حساب Supabase

- اذهب إلى [supabase.com](https://supabase.com)
- أنشئ حساب مجاني
- أنشئ مشروع جديد

### 2. تنفيذ السكربت

- اذهب إلى **SQL Editor** في لوحة Supabase
- انسخ محتوى ملف `supabase-schema.sql`
- الصقه واضغط **Run**

### 3. إضافة المفاتيح

من **Settings → API**، انسخ:

- `Project URL` → ضعه في `NEXT_PUBLIC_SUPABASE_URL`
- `anon key` → ضعه في `NEXT_PUBLIC_SUPABASE_ANON_KEY`

## 🚀 النشر على Vercel

### الطريقة الأسهل:

1. ارفع الكود على [GitHub](https://github.com)
2. اذهب إلى [vercel.com](https://vercel.com)
3. اضغط **Import Project**
4. اختر المستودع
5. أضف متغيرات البيئة
6. اضغط **Deploy**

✅ **خلال دقائق، موقعك سيكون جاهزاً على الإنترنت!**

---

## 🔧 حل المشاكل الشائعة

### المشكلة: `npm: command not found`

**الحل:** ثبّت [Node.js](https://nodejs.org/)

### المشكلة: `npm install` يفشل

**الحل:** حاول:

```bash
npm install --legacy-peer-deps
```

### المشكلة: الصور لا تظهر

**الحل:** هذا طبيعي في وضع التطوير. الصور من Unsplash قد تحتاج اتصال إنترنت.

### المشكلة: لا أرى المنتجات

**الحل:** تأكد أن ملف `lib/mock-data.ts` موجود. الموقع يستخدم بيانات تجريبية افتراضياً.
